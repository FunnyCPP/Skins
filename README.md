## General Information
Application for downloading and installing skins for minecraft. Also user can choose favourites skins.

- Kotlin - version 1.5.10
- Navigation - version 2.3.5
- RxKotlin - version 2.2.0
- Glide - version 4.11.0
- Paperdb - version 2.6
